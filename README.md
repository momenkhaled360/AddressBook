# Address Book
Address Book is a Full Stack application built using:

## Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- Repository Pattern
- Unit Of Work
- AutoMapper
- JWT Authentication

## Frontend
- Angular
- TypeScript
- Tailwind CSS
- Reactive Forms

## Features

### Authentication
- Register
- Login
- JWT Authentication

### Contact Management
- Create Contact
- Update Contact
- Delete Contact
- Get All Contacts
- Search Contacts
- Upload Profile Photo
- Export Contacts To Excel

### Job Management
- Create Job
- Update Job
- Delete Job

### Department Management
- Create Department
- Update Department
- Delete Department
